Getting Started with OpenClaw

OpenClaw is an open-source AI agent framework that lets you build autonomous AI assistants using simple markdown configuration files. With over 150,000 GitHub stars, it has become the most popular platform for personal AI agents.

What is SOUL.md?

SOUL.md is the core configuration file that defines your AI agent's identity, personality, expertise, and behavior rules. A well-written SOUL.md is the difference between a useful agent and a generic chatbot. It includes:

  • Identity ‚Äî Role, expertise, communication style
  • Decision Frameworks ‚Äî How the agent reasons through problems
  • Output Contracts ‚Äî Consistent response formats
  • Memory Rules ‚Äî What to remember and what to forget
  • Safety Boundaries ‚Äî What the agent must never do

Quick Start

# Install OpenClaw
npm install -g openclaw

# Initialize your first agent
openclaw init

# Start the gateway
openclaw gateway start

Your agent is now running. Customize the SOUL.md file to define its behavior, connect a messaging channel (Telegram, Discord, Slack, etc.), and set up scheduled tasks with cron jobs.

Why Use Templates?

Writing a SOUL.md from scratch takes time and experimentation. Templates give you battle-tested patterns that work out of the box. Our free starter pack includes 5 templates for the most common agent types.

Security Matters

AI agents with tool access (shell commands, file system, web browsing) need proper security hardening. The most common mistakes: running as root (60% of deployments), API keys in config files (45%), and no monitoring (80%). Our free deployment checklist covers all 7 security phases.
