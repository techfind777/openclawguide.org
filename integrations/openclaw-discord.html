Setup Guide

Prerequisites

  • OpenClaw installed on your server (Quick install guide)
  • A Discord account with API access or webhook permissions

Step 1: Configure OpenClaw Channel

OpenClaw supports Discord as a native channel. Add the Discord configuration to your OpenClaw config file:

channels:
  discord:
    enabled: true
    # Add your Discord credentials here
