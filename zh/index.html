❓ 常见问题

关于 OpenClaw 的常见疑问

OpenClaw 是什么？+
OpenClaw 是一个开源的 AI 智能体框架，拥有 GitHub 150K+ Stars。它可以让你在自己的服务器上运行 AI 助手，支持 Telegram、Discord、WhatsApp、Slack 等 10+ 聊天平台，完全掌控数据和隐私。
运行 OpenClaw 需要什么配置？+
最低配置：1 vCPU、2GB 内存的 VPS 即可运行。推荐配置：2 vCPU、4GB 内存，可以更流畅地处理多用户并发。需要安装 Docker，大多数 Linux VPS 都支持。
OpenClaw 每月花费多少？+
OpenClaw 本身完全免费开源。主要费用是 VPS 服务器（$5-6/月起）和 AI API 调用费用（如 OpenAI API，按使用量计费）。使用 Vultr 新用户可获得 $300 免费额度，足够使用数月。
可以用免费服务器吗？+
可以。Oracle Cloud 提供永久免费的 ARM 实例（4 vCPU、24GB 内存），非常适合运行 OpenClaw。另外 Vultr 新用户有 $300 免费额度，相当于免费使用数月。不过免费方案通常有性能或地域限制，正式使用建议选择付费 VPS。
