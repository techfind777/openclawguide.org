常见问题

需要技术知识吗？

不需要。只要你会复制粘贴命令，就能使用 EasySetup。脚本会问简单问题，其余全部自动处理。

支持哪些服务器？

任何运行 Ubuntu 22.04+、Debian 12+ 或类似 Linux 的 VPS。已在 Hostinger、DigitalOcean、Vultr、Hetzner 和 Oracle Cloud 上测试。

支持哪些 AI 模型？

Claude (Anthropic)、GPT-4o (OpenAI)、Gemini (Google)、DeepSeek、本地模型 Ollama、OpenRouter（所有模型）。

支持哪些聊天应用？

Telegram、Discord、Slack、WhatsApp、Signal 等。安装时选择。

安装脚本是开源的吗？

是的！你可以在运行前查看完整源代码。我们相信透明。

已经安装了 OpenClaw 怎么办？

脚本会检测现有安装，并提供更新/重新配置的选项，而不是重新安装。

还需要为 AI API 付费吗？

是的，AI 模型是单独的费用（每月 /usr/bin/bash-50，取决于使用量）。查看我们的费用指南。
