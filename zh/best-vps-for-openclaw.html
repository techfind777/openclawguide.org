OpenClaw 服务器最低要求

  • CPU：1核（推荐2核）
  • 内存：2GB（推荐4GB，多 Agent 场景建议8GB）
  • 硬盘：20GB SSD
  • 系统：Ubuntu 22.04+、Debian 12+、CentOS 9+
  • 网络：需要访问 AI API（OpenAI/Anthropic/Google）
💡 提示：OpenClaw 本身很轻量，主要开销在 AI API 调用。一台 $5/月的 VPS 就能跑起来。

快速部署教程

第一步：创建服务器

选择上面任意一家服务商，创建一台 Ubuntu 22.04 或更高版本的服务器。推荐配置：2核 2G 内存。

第二步：SSH 连接

ssh root@你的服务器IP

第三步：一键安装 OpenClaw

npx openclaw

或者使用我们的安装脚本：

curl -fsSL https://openclawguide.org/install.sh | bash

第四步：启动

openclaw gateway start

就这么简单。你的 AI 智能体现在 24/7 运行了。

常见问题

运行 OpenClaw 每月要花多少钱？

VPS：$0-12/月。AI API：$5-30/月（取决于使用量）。总计：大多数用户 $5-42/月。

可以在树莓派上运行吗？

可以，但性能有限。树莓派 4（4GB+ 内存）可以基本使用，树莓派 5 更好。

需要独立 IP 吗？

不需要。任何 VPS 都自带公网 IP。OpenClaw 通过出站连接与消息平台通信，不需要入站端口。

国内服务器能用吗？

能用，但如果要调用 OpenAI/Anthropic 等海外 API，需要配置代理。推荐选择香港或新加坡节点，可以直连海外 API。如果用国内大模型（通义千问等），国内服务器完全没问题。

哪个 VPS 最适合新手？

推荐 Hostinger（最便宜）或 Vultr（最灵活）。两者都有简单的控制面板，5分钟就能创建服务器。
