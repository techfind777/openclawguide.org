Kostenloser Telegram KI-Bot in 5 Schritten

Erstellen Sie einen leistungsstarken KI-Bot auf Telegram in 15 Minuten — mit echten Konfigurationsbeispielen

⏱ 15 Minuten📊 Anfänger📅 Aktualisiert Feb 2026

Warum Telegram + OpenClaw?

Telegram ist eine der besten Plattformen für KI-Agenten. Anders als WhatsApp oder iMessage hat Telegram eine vollständig offene Bot-API ohne Genehmigungsverfahren, ohne Nachrichten-Limits und mit Unterstützung für reichhaltige Formatierung, Inline-Tastaturen und Gruppenchats. In Kombination mit OpenClaws Multi-Agent-Architektur können Sie alles erstellen — von einem persönlichen Assistenten bis zu einem Kundensupport-Bot für Tausende von Nutzern.

Das macht diese Kombination so mächtig:

  • Sofortige Einrichtung — Bot mit BotFather erstellen, einen Konfigurationsblock hinzufügen, fertig
  • Multi-Channel — Derselbe Agent kann gleichzeitig Telegram, Slack, Discord und WhatsApp bedienen
  • Persistentes Gedächtnis — OpenClaw merkt sich den Gesprächskontext über Sessions hinweg
  • Tool-Zugriff — Ihr Bot kann das Web durchsuchen, Dateien lesen, Code ausführen und APIs aufrufen
  • Gruppenchat-Unterstützung — Funktioniert sowohl in DMs als auch in Gruppengesprächen

Voraussetzungen

  • OpenClaw auf einem VPS (Ein-Klick-Installer) oder lokaler Maschine installiert
  • Ein Telegram-Account
  • 5 Minuten freie Zeit

Schritt 1: Telegram Bot mit BotFather erstellen

Öffnen Sie Telegram und suchen Sie nach @BotFather. Dies ist Telegrams offizieller Bot zum Erstellen und Verwalten von Bots.

  1. Senden Sie /newbot an BotFather
  2. Wählen Sie einen Anzeigenamen (z.B. "Mein KI-Assistent")
  3. Wählen Sie einen Benutzernamen, der mit "bot" endet (z.B. mein_ki_assistent_bot)
  4. BotFather gibt Ihnen ein API-Token wie: 7123456789:AAH...
⚠️ Halten Sie Ihr Bot-Token geheim. Jeder mit diesem Token kann Ihren Bot kontrollieren. Falls es jemals geleakt wird, verwenden Sie /revoke in BotFather, um ein neues zu generieren.

Optional aber empfohlen — stellen Sie diese mit BotFather ein:

  • /setdescription — Was Nutzer sehen, bevor sie einen Chat starten
  • /setabouttext — Kurze Bio im Bot-Profil
  • /setuserpic — Geben Sie Ihrem Bot einen Avatar
  • /setcommands — Fügen Sie Befehls-Vorschläge hinzu (z.B. /help, /status)

Schritt 2: OpenClaw konfigurieren

Fügen Sie den Telegram-Kanal zu Ihrer OpenClaw-Konfiguration hinzu. Führen Sie aus:

openclaw config.patch channels.telegram.token "IHR_BOT_TOKEN_HIER"
openclaw config.patch channels.telegram.enabled true

Das ist die Minimalkonfiguration. Ihr Bot ist jetzt verbunden. Aber lassen Sie uns ihn mit ein paar mehr Einstellungen intelligenter machen:

# Bot in Gruppenchats erlauben (optional)
openclaw config.patch channels.telegram.chatTypes '["direct","group"]'

# Welcher Agent Telegram-Nachrichten behandelt
openclaw config.patch channels.telegram.agent "main"

Schritt 3: Persönlichkeit Ihres Agenten definieren

Die Magie von OpenClaw liegt in der SOUL.md-Datei. Diese Markdown-Datei definiert, wer Ihr Agent ist, wie er sich verhält und was er weiß. Hier ist ein echtes Beispiel für einen Telegram-Assistenten:

# SOUL.md — Telegram KI-Assistent

Sie sind Alex, ein hilfreicher KI-Assistent auf Telegram.

## Persönlichkeit
- Freundlich und präzise — Telegram-Nachrichten sollten kurz sein
- Verwenden Sie Emojis sparsam aber natürlich 👋
- Teilen Sie lange Antworten in mehrere kurze Absätze auf
- Verwenden Sie Telegram-Markdown-Formatierung (fett, Code-Blöcke)

## Fähigkeiten
- Fragen zu unserem Produkt/Service beantworten
- Bei technischer Fehlerbehebung helfen
- Erinnerungen und Follow-ups planen
- Das Web nach aktuellen Informationen durchsuchen

## Regeln
- Niemals interne Firmendaten preisgeben
- Rechnungsangelegenheiten an menschlichen Support weiterleiten
- Antworten unter 500 Wörtern halten, außer wenn Details angefragt werden
- Immer in der Sprache des Nutzers antworten

Schritt 4: Bot testen

Starten Sie OpenClaw neu, um die neue Konfiguration zu laden:

openclaw gateway restart

Öffnen Sie jetzt Telegram, finden Sie Ihren Bot über seinen Benutzernamen und senden Sie eine Nachricht. Sie sollten innerhalb weniger Sekunden eine Antwort erhalten.

Wenn etwas nicht funktioniert, prüfen Sie:

  1. openclaw gateway status — Läuft das Gateway?
  2. openclaw doctor — Diagnostik ausführen
  3. Prüfen Sie, dass Ihr Bot-Token korrekt ist (keine zusätzlichen Leerzeichen)
  4. Stellen Sie sicher, dass Ihre VPS-Firewall ausgehende HTTPS (Port 443) erlaubt

Erweitert: Gruppenchat-Setup

Ihren Bot in Telegram-Gruppen zu betreiben erfordert ein paar zusätzliche Schritte:

Gruppendatenschutz-Modus aktivieren

Standardmäßig sehen Telegram-Bots in Gruppen nur Nachrichten, die sie erwähnen oder die mit einem Befehl beginnen. Um Ihrem Bot alle Nachrichten zu zeigen:

  1. Öffnen Sie BotFather
  2. Senden Sie /setprivacy
  3. Wählen Sie Ihren Bot
  4. Wählen Sie Disable (dies ermöglicht dem Bot, alle Gruppennachrichten zu sehen)

Gruppenverhalten konfigurieren

Sie möchten wahrscheinlich nicht, dass Ihr Bot auf jede Nachricht in einer Gruppe antwortet. Konfigurieren Sie Auslöser-Pattern:

# Nur antworten, wenn per Namen erwähnt oder mit @
openclaw config.patch channels.telegram.groupTrigger "mention"

Optionen für groupTrigger:

  • "mention" — Nur antworten wenn @erwähnt (empfohlen)
  • "all" — Auf jede Nachricht antworten (vorsichtig verwenden)
  • "command" — Nur auf /Befehle antworten

Echte Anwendungsfälle

1. Kundensupport-Bot

Richten Sie einen Telegram-Bot ein, der 24/7 Produktfragen beantwortet. Verbinden Sie ihn mit Ihrer Wissensdatenbank und lassen Sie ihn die erste Support-Linie übernehmen. Komplexe Fälle automatisch an menschliche Agenten weiterleiten.

2. Team-Assistent in Gruppenchats

Fügen Sie Ihren Bot zur Telegram-Gruppe Ihres Teams hinzu. Er kann lange Threads zusammenfassen, technische Fragen beantworten, Erinnerungen setzen und Daten aus verbundenen Tools wie Jira oder Notion abrufen.
