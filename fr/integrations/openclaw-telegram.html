Bot Telegram IA Gratuit en 5 Étapes

Créez un bot IA puissant sur Telegram en 15 minutes — avec des exemples de configuration réels

⏱ 15 minutes📊 Débutant📅 Mis à jour Fév 2026

Pourquoi Telegram + OpenClaw ?

Telegram est l'une des meilleures plateformes pour les agents IA. Contrairement à WhatsApp ou iMessage, Telegram a une API Bot entièrement ouverte sans processus d'approbation, sans limites de messages, et avec support pour le formatage riche, claviers inline et chats de groupe. Combiné avec l'architecture multi-agent d'OpenClaw, vous pouvez construire de tout — d'un assistant personnel à un bot de support client servant des milliers d'utilisateurs.

Voici ce qui rend cette combinaison puissante :

  • Configuration instantanée — Créer un bot avec BotFather, ajouter un bloc config, terminé
  • Multi-canal — Le même agent peut servir Telegram, Slack, Discord et WhatsApp simultanément
  • Mémoire persistante — OpenClaw se souvient du contexte de conversation entre les sessions
  • Accès aux outils — Votre bot peut rechercher le web, lire des fichiers, exécuter du code et appeler des APIs
  • Support de chat de groupe — Fonctionne à la fois en messages privés et conversations de groupe

Prérequis

  • OpenClaw installé sur un VPS (installateur un clic) ou machine locale
  • Un compte Telegram
  • 5 minutes de temps libre

Étape 1 : Créer un Bot Telegram avec BotFather

Ouvrez Telegram et cherchez @BotFather. C'est le bot officiel de Telegram pour créer et gérer des bots.

  1. Envoyez /newbot à BotFather
  2. Choisissez un nom d'affichage (ex: "Mon Assistant IA")
  3. Choisissez un nom d'utilisateur finissant par "bot" (ex: mon_assistant_ia_bot)
  4. BotFather vous donnera un token API comme : 7123456789:AAH...
⚠️ Gardez votre token de bot secret. Quiconque avec ce token peut contrôler votre bot. S'il est jamais divulgué, utilisez /revoke dans BotFather pour en générer un nouveau.

Étape 2 : Configurer OpenClaw

Ajoutez le canal Telegram à votre configuration OpenClaw. Exécutez :

openclaw config.patch channels.telegram.token "VOTRE_TOKEN_BOT_ICI"
openclaw config.patch channels.telegram.enabled true

C'est la config minimale. Votre bot est maintenant connecté. Mais rendons-le plus intelligent avec quelques paramètres supplémentaires :

# Permettre au bot de fonctionner dans les chats de groupe (optionnel)
openclaw config.patch channels.telegram.chatTypes '["direct","group"]'

# Définir quel agent gère les messages Telegram
openclaw config.patch channels.telegram.agent "main"

Étape 3 : Définir la Personnalité de votre Agent

La magie d'OpenClaw réside dans le fichier SOUL.md. Ce fichier markdown définit qui est votre agent, comment il se comporte et ce qu'il sait. Voici un exemple réel pour un assistant Telegram :

# SOUL.md — Assistant IA Telegram

Vous êtes Alex, un assistant IA utile sur Telegram.

## Personnalité
- Amical et concis — les messages Telegram doivent être courts
- Utilisez les emojis avec parcimonie mais naturellement 👋
- Divisez les longues réponses en plusieurs paragraphes courts
- Utilisez le formatage markdown Telegram (gras, blocs de code)

## Capacités
- Répondre aux questions sur notre produit/service
- Aider avec le dépannage technique
- Programmer des rappels et suivis
- Rechercher le web pour des informations actuelles

## Règles
- Ne jamais partager de données internes de l'entreprise
- Escalader les problèmes de facturation au support humain
- Garder les réponses sous 500 mots sauf si le détail est demandé
- Toujours répondre dans la langue de l'utilisateur

Étape 4 : Tester votre Bot

Redémarrez OpenClaw pour prendre en compte la nouvelle config :

openclaw gateway restart

Maintenant ouvrez Telegram, trouvez votre bot par son nom d'utilisateur, et envoyez un message. Vous devriez recevoir une réponse en quelques secondes.

Cas d'Usage du Monde Réel

1. Bot de Support Client

Configurez un bot Telegram qui répond aux questions produits 24/7. Connectez-le à votre base de connaissances et laissez-le gérer la première ligne de support. Escaladez automatiquement les problèmes complexes aux agents humains.

2. Assistant d'Équipe dans les Chats de Groupe

Ajoutez votre bot au groupe Telegram de votre équipe. Il peut résumer les longs fils, répondre aux questions techniques, définir des rappels et extraire des données d'outils connectés comme Jira ou Notion.
