OpenClaw Server Requirements

Before choosing a VPS, here's what OpenClaw actually needs:

  • Minimum: 1 vCPU, 1GB RAM, 10GB storage (basic agent, one channel)
  • Recommended: 2 vCPU, 2-4GB RAM, 20GB storage (multiple channels, skills, browser)
  • Heavy use: 4 vCPU, 8GB RAM, 40GB storage (multi-agent, local models, heavy automation)
  • OS: Ubuntu 22.04+ or Debian 12+ recommended
  • Node.js: v22+ (installer handles this)
💡 Pro tip: Most users are fine with 2GB RAM. The AI processing happens on the model provider's servers (Claude, GPT, etc.), not on your VPS. Your server mainly handles the gateway, messaging, and skill execution.

Quick Setup Guide

Once you have your VPS, here's how to install OpenClaw:

Step 1: SSH into your server

ssh root@your-server-ip

Step 2: Run the installer

npx openclaw

Step 3: Follow the setup wizard

The installer will ask for your AI model API key and preferred messaging channel. Have your Claude/OpenAI API key ready.

Step 4: Start the gateway

openclaw gateway start

That's it. Your agent is now running 24/7.

Frequently Asked Questions

How much does it cost to run OpenClaw per month?

VPS: $0-12/month. AI API: $5-30/month depending on usage. Total: $5-42/month for most users.

Can I run OpenClaw on a Raspberry Pi?

Yes, but performance will be limited. A Pi 4 with 4GB+ RAM works for basic use. A Pi 5 is better.

Do I need a dedicated IP?

No. Any VPS comes with a public IP. OpenClaw connects outbound to messaging services — no inbound ports needed (except SSH for management).

Can I run multiple OpenClaw agents on one VPS?

Yes. Each agent uses ~200-500MB RAM. A 4GB VPS can comfortably run 3-5 agents.

Which AI model should I use?

Claude Sonnet 4 is the best balance of quality and cost. Claude Opus 4 for maximum capability. GPT-4o for budget. DeepSeek for cheapest.

Is my data safe on a VPS?

Yes — your data stays on YOUR server. Unlike cloud AI services, nothing is shared with third parties. See our Security Guide for hardening tips.
